## 推荐搜索查询模板

> ⚠️ 所有搜索查询**必须**使用 `site:` 操作符限定官方域名，禁止通用搜索。

### 按公司搜索（首选方式）

```
site:anthropic.com {month} {year}
site:openai.com blog {month} {year}
site:deepmind.google blog {month} {year}
site:ai.meta.com blog {month} {year}
site:mistral.ai news {month} {year}
site:x.ai blog {month} {year}
site:qwen.ai blog {month}
site:zhipuai.cn {month}
site:deepseek.com {month}
site:minimaxi.com {month}
site:moonshot.cn {month}
```

### 按主题搜索（仍需 site: 限定）

```
# 新模型发布
site:{company_official_domain} "new model" OR "model release" {year}

# 开源项目
site:{company_official_domain} "open source" OR "open weight" {year}

# 安全与对齐
site:{company_official_domain} safety alignment {year}

# API 与产品更新
site:{company_official_domain} API update changelog {year}
```

### 学术论文（仅限官方发布）

```
site:github.com/{company_github_org} release {month} {year}
site:huggingface.co/{company_hf_org} {month} {year}
```

> 🚫 **禁止的搜索方式**：
> - ~~"AI model release announcement April 2025"~~（无 site: 限定）
> - ~~"大模型 最新发布 4月"~~（会返回第三方媒体）
> - ~~site:arxiv.org "{company_name}"~~（非官方发布渠道）

## 抓取策略建议

### 优先级排序

1. **第一优先**：Anthropic、OpenAI、Google DeepMind、Meta AI — 行业风向标
2. **第二优先**：DeepSeek、Qwen、智谱 AI — 国内技术领军
3. **第三优先**：Mistral、xAI、月之暗面 — 创新型公司
4. **第四优先**：MiniMax、零一万物、百川等 — 补充覆盖

### 时效性处理

- 搜索查询务必包含时间限定词（月份/年份）
- 优先展示最近 7 天的文章
- 7-30 天的文章作为补充
- 超过 30 天的文章不纳入常规报告

### 去重规则

1. 以官方博客原文 URL 为唯一标识
2. 同一公司关于同一主题的多篇官方文章（如博客+论文+GitHub），合并为一个条目但保留所有官方链接
3. **严禁将第三方媒体转载作为来源**，即使官方原文不可访问，也只标注"官方页面暂不可达"

### 语言处理

- 英文内容翻译为中文摘要，关键术语保留英文
- 中文博客直接引用
- 保留原文链接供读者查阅全文
